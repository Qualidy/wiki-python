    Einführung in Data Science¶

    Begriff Kurzerklärung Link zur Referenz
    Data Science Eine Fachdisziplin, um aus Daten neues Wissen zu generieren Referenz

    Pandas: Einführung¶

    Begriff Kurzerklärung Link zur Referenz
    DataFrame Datenstruktur in Pandas, die eine Tabelle speichert Referenz
    Series Datenstruktur in Pandas, die eine Folge von Werten speichert Referenz
    Index Die Beschriftung der Zeilen einer Pandas Tabelle Referenz
    .loc[] Zugriff auf eine Zeile in einer Pandas Tabelle über Zeilenbezeichnung Referenz
    concat() Zusammenfügen mehreren DataFrames oder Series' Referenz

    Pandas: Import/Export von Daten¶

    Begriff Kurzerklärung Link zur Referenz
    read_csv() Liest eine .csv (comma separated values) Datei ein (eine Tabelle) Referenz
    to_csv() Speichert eine Pandas Tabelle als .csv-Datei Referenz
    to_dict() Wandelt eine Tabelle in ein Python Dictionary um Referenz
    read_pickle() Liest eine Pandas Tabelle aus einer pickle-Datei Referenz
    to_pickle() Speichert eine Pandas Tabelle im pickle-Dateiformat (Binärdatei) Referenz
    to_sql() Speichert eine Pandas Tabelle in einer SQL-Datenbank Referenz
    read_sql_query() Führt einen SQL-Befehl aus einer SQL-Datenbank aus. Ergebnis: Tabelle Referenz

    Pandas: Fehlende Werte¶

    Begriff Kurzerklärung Link zur Referenz
    NaN Steht für "Not a Number". Im Kontext von Tabellen meint dies eine Datenlücke Referenz
    Imputation Einfügen von Werten in Datenlücken Referenz
    isnull() Hilfsfunktion, um fehlende Werte in Pandas DataFrames/Series zu detektieren Referenz
    dropna() Entfernt unvollständige Zeilen oder Spalten in einer Tabelle bzw. Series Referenz
    fillna() Füllt Datenlücken (NaN) mit Werten Referenz
    median() Berechnet den Median über Zeilen oder Spalten (je nach Einstellung) Referenz
    interpolate() Füllt Datenlücken mit Hilfe von benachbarten Werten (Randwerten) Referenz

    NumPy: Einführung und NumPy Arrays¶

    Begriff Kurzerklärung Link zur Referenz
    numpy.array() Ein n-dimensionales Array. Wird sehr effizient gespeichert Referenz
    arange() Erzeugt ein NumPy Array in einem angegebenen Bereich Referenz
    sum() Bildet die Zeilen- oder Spaltensummen eines Arrays Referenz
    max() Berechnet das Zeilen- oder Spaltenmaximum eines Arrays Referenz
    multiply() Multipliziert zwei Arrays elementweise (keine Matrizenmultiplikation!) Referenz
    randint() Erzeugt ein Array mit zufälligen ganzen Zahlen aus einem Intervall Referenz
    transpose() Wandelt Zeilen einer Matrix in Spalten um ("kippt" die Matrix) Referenz

    Numpy: Slicing¶

    Begriff Kurzerklärung Link zur Referenz
    Slicing Bei Python-Listen, NumPy-Arrays oder Tabellen Teile "herrauschneiden" Referenz
    Boolean Indexing Rausschneiden von Teilen eines NumPy-Arrays, das gewissen Bedingungen "genügt" Referenz
    diagonal() Liefert die Diagonalwerte einer Matrix Referenz
    reshape() Wandelt die Werte eines NumPy-Arrays in einer andere Form (Dimensionalität) um Referenz

    Matplotlib: Linienplots¶

    Begriff Kurzerklärung Link zur Referenz
    plot() Plottet (x,y)-Punkte mit Markern und/oder verbindet diese mit Linien (Linienplot) Referenz
    Marker Visualisiert einzelne Datenpunkte über Kreise, Dreiecke, Sterne, etc. Referenz
    annotate() Erlaubt es Anmerkungen zu einzelnen Datenpunkte über Pfeile+Text hinzuzufügen Referenz
    plt.legend() Fügt einem Diagramm eine Legende zu, die zeigt, was die einzelnen Linien bedeuten Referenz
    plt.grid() Zeigt Gitterlinien im Diagramm an. Erhöht die Lesbarkeit Referenz

    Matplotlib: Subplots¶

    Begriff Kurzerklärung Link zur Referenz
    Subplot Ein Unterdiagramm in einem großen Diagramm Referenz
    plt.subplot() Fügt einen neuen einzelnen Subplot (Unterdiagramm) hinzu Referenz
    plt.subplots() Erzeugt eine ganze Menge (ein Gitter) von Subplots Referenz
    tight_layout() Optimiert die Abstände zwischen und um Subplots herum Referenz
    subplots_adjust() Erlaubt es horizontale und vertikale Abstände zwischen Subplots zu definieren Referenz
    subplot2grid() Erlaubt es Subplots über mehrere Zeilen/Spalten des Subplotgitters zu erzeugen Referenz

    Seaborn: Einführung¶

    Begriff Kurzerklärung Link zur Referenz

    countplot() Balkendiagramm, das zeigt, wie oft bestimmte Kategorien auftreten Referenz
    boxplot() Visualiert Median, Quartile, Spannweiter und Ausreißer in einem Diagramm Referenz

    jointplot() Kombination von Scatterplot und Histogramm-Plots Referenz
    kdeplot() Zeigt Verteilungen von Daten über Isokonturlinien ("Höhenlinien") an Referenz

    Matplotlib: Ticks¶

    Begriff Kurzerklärung Link zur Referenz
    Ticks (bei Matplotlib Diagrammen) Ticks sind die Markierungen auf den Achsen eines Matplotlib-Diagramms, die die Skalenwerte repräsentieren Referenz
    xticks() Definiert, wo die Ticks auf der x-Achse gesetzt werden Referenz
    yticks() Definiert, wo die Ticks auf der y-Achse gesetzt werden Referenz
    xscale() Setzt die Skala (linear, logarithmisch) für die x-Achse Referenz
    yscale() Setzt die Skala (linear, logarithmisch) für die y-Achse Referenz
    tick_params() Erlaubt es z.B. die Farbe der Ticks zu ändern Referenz

    Pandas & Matplotlib: Zeitreihenanalyse¶

    Begriff Kurzerklärung Link zur Referenz
    Zeitreihe Eine zeitlich geoordnete Folge von Werten, Bsp: Sensorwerte oder Umsatzzahlen Referenz
    rolling() Erzeugt ein gleitendes Fenster, z.B. um einen gleitenden Mittelwert zu berechnen Referenz
    resample() Erzeugt eine neue zeitliche Sicht auf eine Zeitreihe Referenz
    Lineare Regression Verfahren, das Datenpunkte versucht durch eine Gerade zu beschreiben Referenz
    scikit-learn Python Bibliothek mit sehr vielen Machine-Learning-Verfahren, u.a. Lineare Regression Referenz
    LinearRegression Klasse aus scikit-learn, um eine lineare Regression zu lernen Referenz

    Pandas: Outlier (Ausreißer)¶

    Begriff Kurzerklärung Link zur Referenz
    IQR Der Interquartilbereich (IQR) ist die Differenz zwischen dem 75. und 25. Perzentil eines Datensatzes und wird verwendet, um Ausreißer zu identifizieren Referenz
    Z-Score Der Z-Score ist eine statistische Maßzahl, die angibt, wie viele Standardabweichungen ein Datenpunkt vom Mittelwert der Datenmenge entfernt ist Referenz
    Winsorizing Winsorizing ist eine Methode zur Reduzierung von Ausreißern, indem extrem hohe und niedrige Werte auf einen bestimmten Perzentilwert gesetzt werden Referenz
    Quantil Ein Quantil ist ein Wert, der einen Datensatz in gleich große, aufeinanderfolgende Teile teilt, wobei das 50. Perzentil beispielsweise dem Median entspricht Referenz
    Standardabweichung Die Standardabweichung ist ein Maß für die Streuung der Werte in einem Datensatz, wobei höhere Werte auf eine größere Variabilität hinweisen Referenz
    zscore() Funktion zur Berechnung des Z-Scores Referenz
    quantile() Berechnet Quantile, z.B. zur Berechnung des 95% Perzentils Referenz
    winsorize() Funktion zur Begrenzung von Ausreißern Referenz
